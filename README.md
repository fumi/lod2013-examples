# lod2013-examples

LODチャレンジ2013 第6回LODチャレンジデー用のサンプルコード

## 事前準備

* Ruby-1.9.2以上 or Ruby-2.0.0以上

## 使い方

### bundlerインストール
    $ gem install bundler


### サンプルコードインストール

    $ git clone https://github.com/fumi/lod2013-examples.git
    $ cd lod2013-examples/src/
    $ bundle install

### サンプル起動

    $ bundle exec shotgun app.rb

ブラウザで http://localhost:9393 見らればOK．
